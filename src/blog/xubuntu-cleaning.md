Будем очищать через терминал Xubuntu:

Очищаем систему от .deb пакетов, которые более не нужны:

``` bash
sudo apt-get autoclean
```


Удаляем ненужные нам зависимости от уже удалённых пакетов:
``` bash
sudo apt-get autoremove

```
